# Gestor de Vida

## Description

Gestor de Vida is an internal application for mananing the life of a person: tasks, projects, calendar, todos, habits, finance, etc.

For it it would use the web and a native app for linux for general managing of all, and then for fast manipulation it will use an android app 

## Tech stack overview

- [Psql](https://www.postgresql.org/docs/current/app-psql.html) for database 
- [FastAPI](https://fastapi.tiangolo.com/) for the backend API 
- [Sveltekit](https://kit.svelte.dev/) for the frontend
- [Tailwindcss](https://tailwindcss.com/) for the frontend
- [Tauri](https://tauri.app/) for the native app
- [Svelte-Kit-Tauri](https://github.com/sveltejs/kit/tree/master/packages/kit-tauri) for the tauri integration
- [Kotlin](https://kotlinlang.org/) for the android app
- [Jetpack Compose](https://developer.android.com/jetpack/compose) for the android app
